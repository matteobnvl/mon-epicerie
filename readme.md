## Installation projet

- ``` git clone https://github.com/matteobnvl/mon-epicerie.git ```

- ``` cd mon-epicerie ```

- ``` composer install ```

## lancer le serveur

- ``` symfony server:start ```

## lister les serveurs en cours 

- ``` symnfony server:list ```

## stopper le server 

- ``` symfony server:stop ```
